Méthodologie : comment utiliser cette page ? Guide reps

Cartes interactives ci-dessous : clique pour dérouler les détails d’usage dans ton cycle de prospection.

1
Charger la base CSV Indeed
📂
Dépose le fichier advertisers_actifs_france_avec_EBA.csv exporté d’Indeed FR.
Clique sur « Charger un CSV Indeed France » ci-dessous, sélectionne le fichier advertisers_actifs_france_avec_EBA.csv (France uniquement) puis laisse la page recalculer automatiquement les scores EBA par secteur. Le CSV reste local à ton navigateur.
2
Identifier les comptes mûrs EBA
🔍
Utilise le score EBA, le secteur, la taille, le canal et les tags pour repérer les comptes à fort potentiel display.
Score basé sur : intensité estimée des campagnes social, richesse probable des contenus EBA (taille/secteur) et dynamique de recrutement (proxy via budget). Les tags indiquent les axes clés : profils pénuriques, campagnes nationales, image institutionnelle, etc.
3
Préparer le pitch EBA
💬
Appuie-toi sur la colonne « Commentaire pitch EBA » pour ton argumentaire.
Le commentaire résume l’angle : soutien aux plans de recrutement, visibilité continue, complémentarité avec LinkedIn et social. Adapte avec tes infos terrain (pénuries, priorités métiers, régions).
4
Vérifier les signaux en temps réel
🔗
Ouvre la librairie publicitaire LinkedIn et cherche le compte avant un call.
Clique sur « LinkedIn Ad Library » dans la colonne Liens : tu arrives sur la librairie publicitaire. Tape ensuite le nom du client (ex. « Thales France », « Ville de Paris ») pour vérifier les dernières campagnes EBA.

Légende score EBA (0–100)

  • 0–40 : signaux EBA faibles ou émergents
  • 40–70 : EBA en développement (potentiel à structurer)
  • 70–100 : EBA très active, fort potentiel EBA Indeed (à activer / prioriser)

Score calculé sur : (1) intensité/nature estimée des campagnes social & display (proxy via investissement recrutement), (2) richesse probable des contenus EBA web (proxy via taille & secteur), (3) dynamique de recrutement visible. Saisonnalité EBA basée sur signaux publiés 2022–2025 quand disponibles, sinon sur des tendances sectorielles.
